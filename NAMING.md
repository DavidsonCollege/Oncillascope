# App naming — candidates & trademark/availability vetting

**Purpose:** the working name "AirScope" is already taken on the Mac App Store (see below),
so the app needs a new name before an App Store release. This document tracks vetted
candidates so we can pick one.

**Method:** each name was checked for (a) an existing Apple **App Store** app of the same
name, and (b) a conflicting **company / trademark / brand** via public web search.

> ⚠️ **Not legal advice.** These are preliminary public-web searches, not a formal
> clearance. Before filing/launching, run a full USPTO clearance search (or have a
> trademark attorney vet it) in the relevant classes — **IC 009** (software) and
> **IC 042** (SaaS). Last updated: 2026-06-26.
